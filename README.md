### Installation
- Python 3.8+
- Django

- Clone the repository
~~~
 git clone https://github.com/jerinpious/Django-Portfolio.git
 cd portfolio
~~~
- Install the required packages:
~~~
pip install -r requirements.txt
~~~
- Run the Django server:
 ~~~
 python manage.py runserver
 ~~~
